指針類型

Introduction

  1. 變量預設存的為”值“，稱值類型或是普通類型
  2. 使用&獲取變量的地址，例:var a int，要獲取a的地址就是&a
  3. 指針類型，變量存的是地址，這個地址存的才是值
  4. 獲取指針類型所指向的值，使用*，比如:var *p int, 使用*p獲取p指向的值
